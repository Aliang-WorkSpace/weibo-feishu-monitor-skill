# Share This Skill

This directory is the shareable package for the `weibo-feishu-monitor` skill.

## Give It To Someone Else

Ask the recipient to copy `weibo-feishu-monitor/` into:

```bash
~/.codex/skills/
```

Result:

```bash
~/.codex/skills/weibo-feishu-monitor
```

Then have them read:

- `weibo-feishu-monitor/SKILL.md`
- `weibo-feishu-monitor/references/distribution.md`
- `weibo-feishu-monitor/references/commands.md`

## Important

This package contains the skill only. It does not contain:

- your live Weibo cookies
- your Feishu credentials
- your production monitor source repo
- your launchd configuration

The recipient must map the skill onto their own repo, deployment path, scheduler, and secrets.
